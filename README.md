# Fire Player

A simple video player with functionality such as playing, pausing and stopping a video, as well as minimizing the application to the system tray area.

![fireplayer screenshot_1](/bitmaps/screenshots/screen_1.png) ![fireplayer screenshot_2](/bitmaps/screenshots/screen_2.png) ![fireplayer screenshot_3](/bitmaps/screenshots/screen_3.png) 

## Language

Python

## Modules used

* [wxPython](https://www.wxpython.org/)
* [MPlayer](http://www.mplayerhq.hu)
* [MplayerCtrl](https://pypi.org/project/MplayerCtrl/)
